# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.0.1   | :white_check_mark: |


## Reporting a Vulnerability

To report a vulnerability make an issue with the fallowing template:

**TITLE**

Vuln: [Vulnerability Title]

---

**CONTENT**

Issue: [short summary]

[Filename] [Line]:
```
code in question
```

Solution: [short explanation how it can be fixed]
```
Possible solution
```
